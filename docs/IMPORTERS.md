# Importers

NexusIntel supports preview-first import paths for Maltego CSV, SpiderFoot CSV, Generic IOC CSV, GHunt/Holehe/Maigret/Sherlock JSON, ExifTool JSON, Nmap XML, Amass/Subfinder output, URLScan/Shodan/Censys/VirusTotal JSON exports, blockchain CSV, browser bookmarks HTML, saved pages, and screenshot manifests. Imports are stored as evidence before graph promotion.
