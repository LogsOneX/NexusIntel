# Local-Free OSINT Strategy

NexusIntel is local-first: local analysis, public passive sources, browser-assisted tasks, analyst-imported evidence, open-source tool imports, local graph intelligence, and local AI/rules reasoning work before optional BYOK connectors.

Paid APIs are accelerators only. Candidates are never facts. Every finding is classified as verified, strong, probable, weak, candidate, noise, contradicted, or insufficient_evidence.
