# Legacy Archive

Gunakan `main.py` di root untuk workflow OSINT terintegrasi.

- `legacy/nexusrecon/` - arsip versi username enumeration lama.
- `legacy/nexusrecon_ultimate/` - arsip versi email/phone/domain ramo.
